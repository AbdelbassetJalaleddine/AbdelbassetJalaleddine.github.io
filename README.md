# Abdelbasset Jalaleddine — Portfolio

## 🚀 How to Deploy on GitHub Pages (Free)

### Step 1 — Add your images
Add your images to the `img/` folder and your `cv.pdf` to the root.

### Step 2 — Create a GitHub account
Go to https://github.com and sign up.

### Step 3 — Create a new repository
1. Click **+** → **New repository**
2. Name it: `yourusername.github.io` (use your actual username)
3. Set to **Public** → **Create repository**

### Step 4 — Upload your files
1. Click **uploading an existing file**
2. Upload ALL files keeping folder structure intact
3. Click **Commit changes**

### Step 5 — Enable GitHub Pages
1. Go to repo → **Settings** → **Pages**
2. Source: **Deploy from a branch** → branch: **main** → folder: **/ (root)**
3. Click **Save**

Your site will be live at: `https://yourusername.github.io`
